Powerful Features

More than 400 tools, utilities and applications comprising of more than one million lines of source code

Includes interactive 3D solid geometry editing, high-performance ray-tracing, network-distributed framebuffer support, image and signal-processing tools, path-tracing and photon mapping, a system performance analysis benchmark suite, an embedded scripting interface and libraries for robust high-performance geometric representation and analysis.

Free & Open

Freedom to see the source code, learn from it and improve it

The code is licensed under Open Source Initiative approved license terms. You can see what every line does, customize it and share your work with the community. Contributions from the community make this one of the best CAD programs in the world. Moreover this software, along with updates and support will always be free of charge.

Since 1979

Proudly one of the oldest Open Source repositories in the world

Mike Muuss began the initial architecture and design of BRL-CAD back in 1979. Development as a unified package began in 1983. The first public release was made in 1984. BRL-CAD became an open source project on 21st December 2004.
